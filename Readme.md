
## Prerequsitions

We currently use .net 6. Download from here: https://dotnet.microsoft.com/download/dotnet/6.0

## Getting Started

In backend part 

- Open Visual Studio 

- Run in Package Manager Console
```
update-database
```


## For run backend 

in folder Dolist.Api
```
dotnet run 
```

## For run frontend

in folder Dolist.Client/dolist
```
npm install
npm run start
```
